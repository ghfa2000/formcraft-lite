== Description ==
A lightweight, shortcode-powered contact form plugin with AJAX support and admin settings. Inspired by Contact Form 7, built for speed and simplicity.

== Installation ==
1. Upload the `formcraft-lite` folder to `/wp-content/plugins/`
2. Activate the plugin via the Plugins menu
3. Add `[formcraft_form]` with nested shortcodes to any page

== Shortcodes ==
- `[formcraft_form]` ... `[/formcraft_form]`
- `[text name placeholder="Your Name"]`
- `[email email placeholder="Your Email"]`
- `[textarea message placeholder="Your Message"]`
- `[submit "Send"]`
